# To-Do List Application

## Description
A simple Python-based To-Do List application that allows users to add and view tasks.

## Features
- Add tasks
- View tasks
- User-friendly menu

## Technologies Used
- Python

## How to Run
1. Open terminal
2. Run:
   python todo_list.py

2. View Tasks
